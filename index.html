Analizador Lexico

Um analisador lexical ou lexicográfico é a primeira fase de um compilador, que consiste num programa que recebe como entrada o código fonte de outro programa (sequência de caracteres) e produz uma saída composta de fichas. Um token é um componente ou palavra lexical. O analisador léxico tem de procurar estes tokens a partir do programa fonte.
